Plug&Go Project Overview
Plug&Go is a location-based web (or mobile) application that helps users find nearby electric vehicle (EV) charging stations in real time. 
It focuses on convenience, accessibility, and eco-friendly mobility.

Objectives
Help EV owners quickly find charging stations near their current location.
Show details like availability, charging speed, cost, and contact info.
Provide a clean and user-friendly map-based interface.
Encourage the adoption of electric vehicles by making charging easier.
